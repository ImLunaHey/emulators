Goal: let users play commercial PS1 discs without supplying their own BIOS ROM.

**Decision (2026-06-14): hybrid OpenBIOS-first**, not pure-Rust HLE. Bundle PCSX-Redux's MIT-licensed OpenBIOS as `openbios.bin` (512 KB) and feed it through the EXISTING real-BIOS path (`psx.load_bios`, `Ps1Player.tsx`) — near-zero core work. Pure-Rust HLE BIOS becomes a later, optional fallback track for titles OpenBIOS can't run.

**Why:** satisfies the goal with far less effort than reimplementing the kernel in Rust, and reuses code that already works.

**Key caveat:** OpenBIOS commercial-game compatibility is its weak spot (built mainly for homebrew). So the plan is **validation-first**: build openbios.bin, boot `public/Spyro the Dragon (USA).bin` (already in repo), and only productionize the build/CI/bundling if it actually plays.
